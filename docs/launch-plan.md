## Ключевые риски

| Риск                                            | Вероятность | Последствие                         | Митигация                                              |
| ----------------------------------------------- | ----------- | ----------------------------------- | ------------------------------------------------------ |
| Отсутствие ОКЭД                                 | Средняя     | Штраф за деятельность без ОКЭД      | Добавить сегодня через egr.gov.by                      |
| Нет уведомления в НЦЗПД                         | Средняя     | Предупреждение, предписание         | Подать одновременно с ОКЭД                             |
| Ошибки при миграции Supabase → PostgreSQL       | Средняя     | Баги в auth, потеря данных          | Тесты на локальном PostgreSQL перед деплоем на VPS     |
| Downtime при переключении DNS                   | Низкая      | Несколько часов недоступности       | Снизить TTL заранее, переключать ночью                 |
| Отсутствие регистрации в БелГИЭ при монетизации | Высокая     | Блокировка ресурса                  | Зарегистрироваться до первой оплаты                    |
| WEBPAY не поддерживает Mir для цифровых услуг   | Средняя     | Нет возможности принимать Mir-карты | Звонок в WEBPAY до начала разработки платёжного модуля |
| Перегрузка CloudVPS-1 при росте                 | Низкая      | Деградация производительности       | Апгрейд до CloudVPS-2 (без миграции данных)            |
