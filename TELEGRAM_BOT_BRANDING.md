# Telegram Bot Branding & Configuration

## App Name Recommendation

### 🏆 Chosen: **Clario**

> _Clario_ — clarity + learn. Clean, modern, and immediately communicates the goal: understanding.

**Why it works:**

- Sounds professional and approachable in both English and Russian ("Клэрио")
- Rooted in _clarity_ — the feeling after a concept finally clicks
- Works naturally as a SaaS brand: `@ClarioBot`, `clario.app`, etc.
- Tagline writes itself: _"Learn anything. Understand everything."_

---

## BotFather Configuration

Paste each block exactly as shown into @BotFather.

### 1. Bot Display Name

`/setname`

```
Clario
```

---

### 2. Bot Description

`/setdescription`
_(shown on the bot profile page and when a user opens the bot for the first time — up to 512 chars)_

```
Clario turns anything you read, watch, or paste into smart flashcards — instantly.

Drop in a YouTube link, a PDF, an article, or just a topic name. Clario extracts the key ideas and builds a set of focused study cards for you.

Study them right here in Telegram. No app install. No friction.

Built for people who want to actually remember what they learn. 🧠
```

---

### 3. About Text

`/setabouttext`
_(shown in the contact card preview — up to 120 chars)_

```
AI flashcards from any content. Drop a link or topic — get smart study cards. Study in Telegram. 🧠
```

---

### 4. Command List

`/setcommands`
_(defines the autocomplete menu when users type `/`)_

```
start - Open the app
help - What can Clario do?
```
